## 🤖 What I'm Building

I'm building **[babel-system](https://github.com/desireem-seb/babel-system)** — a collection of open-source AI-powered marketing tools for modern marketing teams.

The goal: lightweight, AI-native tools that handle the structural, repetitive work of marketing so teams can focus on strategy.

---

## 🧰 Projects

### 📊 [Campaign Builder](https://github.com/desireem-seb/babel-system/tree/main/campaign-builder-public)
An open-source campaign framework builder for B2B and SaaS marketing teams.

- Define messaging frameworks, personas, and channels per product
- Manage a full asset repository with AI-powered content generation
- Plan campaigns by quarter and map buyer journeys visually
- Built with Node.js + Claude API · MIT License · Free to use

> *"Marketing is hard enough. This handles the structure."*
